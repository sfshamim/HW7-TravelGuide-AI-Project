# HW7-TravelGuide-AI-Project
This Travel Guide take Input from end user as:

    •    Destination to Travel
    •    Number of Days to be spent there
    •    Special Interests (like Museums, Food & Cuisine, Historic sites, Nightlife, Nature, Shopping etc)
    •    Guardrail to be included (e.g., No walking tours, only kids friendly activities, only wheelchair accessible places etc)

It then generates the Output as following:

    •    Generate a complete plan of places to visited during whole trip, divided by each day.
    •    Generate a PDF of Travel Plan
    •    Include a “Reset Form” Button to reset all fields, so that new plan can be generated.
